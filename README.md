# pattern-recognition
虚拟手物交互是在三维场景中构建人手骨骼模型和交互对象模型，根据静态手势分类结果实现对交互对象的一系列操作，其中手势动作通过终端摄像头基于深度学习识别，三维场景基于Babylon.js图形引擎搭建。该项目还在功能完善阶段，目前实现了手势控制物体平移、旋转和缩放的交互功能，未来将加入装配体模型的拆分、重组、线框化等交互功能，敬请期待。
